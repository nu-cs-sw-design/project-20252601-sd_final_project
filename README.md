# Project: F1 Lap Analyzer

## Contributors
Emily Hu

## Dependencies
- React (19.2.1) -> Frontend
- Node.js (v24.11.1), npm (11.6.2), Vite (vite@7.2.2) -> Build and runtime
- Javascript, Typescript
- FastF1 API

## Build Instructions
Install dependencies with: npm install

Run a development server wtih: npm run dev
